实现手机屏幕替换电脑鼠标

禹穆穆2025/9/2...大约 2 分钟win11

适用场景

  • 没有鼠标
  • 有鼠标但无桌面导致鼠标光标移动不准
  • 简单远程控制

原理

  • 通过node.js的nut.js库完成对鼠标光标的操作，数据通信使用socket.io
    • 注：nut.js可以替换为robot.js，但robot.js基于c++编译正常运行需要安装Microsoft Visual Studio，详情可查看官网，本人弃用robot.js
  • 如果需要在手机屏幕显示电脑屏幕画面，需要使用node.js的screenshot-desktop库完成画面传输

实现

所需环境

  • Node.js 14 +
  • pnpm包管理器

安装依赖

  • 创建一个文件夹（如control-screen），运行pnpm init
  • 分别安装nut.js、express、screenshot-desktop
pnpm i @nut-tree/nut-js express screenshot-desktop

服务端

  • 新建server.js文件，代码如下：
const express = require('express');
const app = express();
const http = require('http').createServer(app);
const io = require('socket.io')(http);
const screenshot = require('screenshot-desktop');
const { mouse, screen } = require('@nut-tree/nut-js');

app.use(express.static('public'));
app.get('/', (req, res) => {
  res.sendFile(__dirname + '/public/index.html');
});

async function getScreenSize() {
  const width = await screen.width();
  const height = await screen.height();
  return { width, height };
}

const handleScale = (phoneX, phoneY,screenWidth, screenHeight) =>{
  //此次可调整比例
  return {
    scaleX: parseFloat(screenWidth/phoneX + 0.5).toFixed(2),
    scaleY: parseFloat(screenHeight/phoneY + 0.5).toFixed(2),
  }
}

io.on('connection', (socket) => {
  console.log('***************  手机端已连接  ***************')
  try{
    const screenshotInterval = setInterval(() => {
      screenshot().then((img) => {
        socket.emit('screenshot', img.toString('base64'));
      });
    }, 200);

    socket.on('mouse-move', async (data) => {
      const { width, height } = await getScreenSize();
      const { scaleX, scaleY } = handleScale(width, height, 2560, 1600);
      console.log('宽高和比例', width, height, scaleX, scaleY);
      data.x = data.x * scaleX;
      data.y = data.y * scaleY;
      console.log('经过比例换算后的坐标：', data);
      await mouse.setPosition(data);
      console.log('***************')
    })

    socket.on('mouse-click', async () => {
        await mouse.leftClick();
      })

    socket.on('disconnect', () => {
      console.log('***************  手机端已断开  ***************')
      clearInterval(screenshotInterval);
    });
  }catch (error) {  
    console.error(error);
  }
});

http.listen(3000, () => {
  console.log('***************  服务已启动  ***************')
})

客户端

  • 在control-screen目录下新建public文件夹
  • 在publick文件夹中新建index.html文件，代码如下：
<!DOCTYPE html>
<html>
<head>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>手机远程控制</title>
  <script src="https://cdn.socket.io/4.5.4/socket.io.min.js"></script>
  <style>
    #screen { width: 100%; height: 80vh; background: #000; }
  </style>
</head>
<body>
  <div id="body" style="width: 100%; height: 100%;">
    <img id="screen"  src="`data:image/png" />
  </div>
  <script>
    const socket = io('http://192.168.127.113:3000');
    const screen = document.getElementById('screen');

    // 接收屏幕截图并显示
    socket.on('screenshot', (img) => {
      console.log("截图地址：",img);
      screen.src = `data:image/png;base64,${img}`;
    });

    // 监听触控事件
    screen.addEventListener('touchmove', (e) => {
      e.preventDefault();
      const rect = screen.getBoundingClientRect();

      //cursor.innerText = `rect:${rect.left}, ${rect.top}, ${rect.width}, ${rect.height}`;
      const x = (e.touches[0].clientX - rect.left) * (window.screen.width / rect.width);
      const y = (e.touches[0].clientY - rect.top)  * (window.screen.height / rect.height);
      socket.emit('mouse-move', { x, y });
    });

    screen.addEventListener('touchend', () => {
      socket.emit('mouse-click');
    });
  </script>
</body>
</html>

查看效果

  • 运行server.js文件，node server.js
  • 手机端打开浏览器，输入http://192.168.xxx.xxx:3000，建议横屏操作

注意点

  • 电脑和手机处于统一局域网内，远程可以通过内网穿透获取域名地址，替换上述代码中url部分
  • 手机屏幕尺寸和电脑屏幕尺寸的比例映射需要单独调整
  • 本篇仅仅实现简单了鼠标操作，远程画面相关可以结合vue或react相关技术栈完善
