win11开机自动执行脚本文件

禹穆穆2025/3/16...大约 2 分钟win11

前言

  • 为了实现在vscode中根据日落时间自动切换暗色和亮色主题，前后找了两种方案，一种是VScode插件方式实现，另一种是通过系统自动执行脚本文件修改vscode配置文件的主题字段，以采用前者，但对后者进行了抽象总结，不实现切换主题效果，但提供其他案例的js脚本（这里可以替换为其他语言脚本）

准备js脚本文件（案例）

  • 功能说明：打开vscode及其他多个应用
  • 代码
const { exec } = require('child_process');

// 应用完整整路径

const pathList = [
    "D:\\VSCode\\Code.exe"
]


// 执行命令来打开应用
const openApp = (path) => {
  try {
    exec(path, (error, stdout, stderr) => {
      if (error) {
          console.error(`执行命令时出错: ${error.message}`);
          return false;
      }
      if (stderr) {
          console.error(`命令执行过程中出现错误: ${stderr}`);
          return false;
      }
      console.log(`应用程序已成功打开: ${stdout}`);
    });

  } catch (error) {
    console.log("出现错误：", error)
  }
}

// 打开应用
for (let i = 0; i < pathList.length; i++) {
    openApp(pathList[i])
}

win11自动执行脚本文件

  • 在开始菜单旁边的搜索中查找任务计划程序并打开
  • 在任务计划程序库中，新建一个自己的文件夹（我的是myplan）：右边操作栏-新建文件夹
  • 点击myplan右键，选择创建任务
  • 给任务命名：常规-名称，命名openMyApp
  • 触发器：什么时候触发，新建-开始任务设为登录时，点击确定
  • 操作：由于js文件需要在Node环境中运行，选择node.exe文件的位置，在参数中把之前写好的脚本文件url复制进去，点击确定
  • 其他设置，可以选择设置超时时间等条件，此步可跳过
  • 点击确定
