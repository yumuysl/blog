利用内网穿透查看本地项目

禹穆穆2025/3/16...大约 3 分钟内网穿透

内网穿透原理

什么是内网穿透

内网穿透（Network Penetration），又称为NAT穿透（NAT Traversal），是一种使位于内网或防火墙后的设备能够被外网访问的技术。通常情况下，位于内网的设备无法直接被外网访问，因为内网使用了私有IP地址，而这些地址在互联网中不可路由。内网穿透技术通过创建隧道，将内网设备的端口暴露到公网，使得外部设备可以通过公网IP地址访问内网服务。

内网穿透原理

  1. 客户端和服务器通信：内网设备（客户端）与具有公网IP的服务器建立连接，创建一个隧道。
  2. 隧道传输：外部设备通过访问服务器的公网IP和指定端口，与内网设备通信。
  3. 数据转发：服务器将收到的数据通过隧道转发给内网设备，完成通信。

使用场景（AI总结）

  1. 远程办公和远程桌面：员工可以通过内网穿透技术在家中或其他地点访问公司内网服务，实现远程桌面操作、文件共享和其他办公需求。

  2. 物联网（IoT）设备的远程监控：许多物联网设备需要通过互联网进行远程监控和管理，内网穿透技术可以帮助实现这一目标。

  3. 家庭网络服务：家庭用户在搭建家庭服务器、NAS存储设备或本地游戏服务器时，可能会面临内网设备无法被外部访问的问题，内网穿透技术可以解决这个问题。

  4. 视频监控和安全系统：内网穿透技术可以使用户通过手机或PC等设备访问家庭或公司内部的监控摄像头，实现实时的远程监控和报警系统。

  5. 个人博客和Web应用服务：开发者在本地搭建个人博客、网站或测试Web应用时，内网穿透技术可以使外界访问这些服务。

  6. 游戏主机和服务器：内网穿透技术适用于游戏主机或私有游戏服务器的设置，使得玩家可以在不同地点进行联机游戏

常见内网穿透工具

一些常见的内网穿透工具包括Ngrok、FRP和ZeroTier。这些工具可以帮助用户在没有公网IP的情况下，实现内网设备与外部设备的通信。

无服务器无域名，通过Ngrok实现移动端查看PC端本地项目文档

  • 当身边没有五服务器也没有域名但需要多人查看自己电脑上的项目时，除了投影方式以外，还可以通过Ngrok以内网穿透形式获取临时url方便他人查看项目
  • 步骤：
    • 在Ngrok官网注册账号，获取authtoken
    • 下载(https://ngrok.com/downloads/windows) Ngrok.exe压缩包，非windows系统参考官网方式下载
    • 解压到目标位置，双击gnrok.exe打开终端，在终端输入：ngrok config add-authtoken <TOKEN>
    • 上一步如果有问题，可以在查看是否有错误码，到官网(https://ngrok.com/docs/errors/reference/)查看对应错误及解决方案
    • 开启本地项目，假设本地项目端口号为8080
    • 终端输入：ngrok http http://localhost:8080
    • 复制获取的连接地址分享他人打开查看
    • 其他事项：
      • 在他人查看项目时，本地ngrok终端服务不能关闭，因为这意味着停止服务，他人不可查看
      • 在后续再次打开ngrok获取地址时，会发现地址和上一次的地址域名不同，如果需要固定域名，需要在ngrok平台面板中domains菜单中创建一个域名，然后终端输入ngrok http 8080 --url <your url>
