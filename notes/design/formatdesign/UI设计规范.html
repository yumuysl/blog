几个常用且通用的交互设计原则

常用的交互设计原则包括：

  1. 可用性：设计应该以用户为中心，注重用户的需求和期望。界面应该简单、直观，并且易于导航和操作。

  2. 一致性：保持界面的一致性，使用户在整个应用程序或网站中能够轻松理解和使用。一致的布局、导航和交互模式可以帮助用户建立模式并提高效率。

  3. 反馈：提供及时和明确的反馈，以告知用户他们的操作是否成功或失败。反馈可以是视觉上的（如状态指示器或动画效果）、听觉上的（如声音提示）或触觉上的（如震动反馈）。

  4. 易学性：设计应该易于学习，新用户能够快速上手并理解如何使用应用程序或网站。简化复杂操作、提供明确的指导和帮助文档可以提高易学性。

  5. 可访问性：设计应该考虑到各种用户的需求，包括身体上的残障、认知能力和技术能力。提供可访问性选项，如调整字体大小、使用屏幕阅读器等，以确保所有用户都能够访问和使用界面。

  6. 可预测性：设计应该符合用户的期望和常规模式。遵循常见的设计模式和标准，减少用户的认知负担，并提供预测性的交互体验。

  7. 简洁性：保持界面的简洁和精简，避免过多的元素和复杂的布局。通过去除冗余和不必要的内容，使用户能够专注于关键任务和信息。

  8. 可扩展性：设计应该具有可扩展性，能够适应未来的需求和变化。考虑到应用程序或网站的可扩展性，以便能够添加新功能或适应不同的设备和平台。

这些交互设计原则可以帮助设计师创建具有良好用户体验的界面。然而，具体的设计原则可能因项目的特定需求、目标用户和平台而有所不同。

移动端设计规范

基本概念
  1. 屏幕密度：DPI和PPI DPI（每英寸点数）和PPI（每英寸像素数）是两个与图像和打印相关的术语。
  • DPI（每英寸点数）指的是打印设备（如打印机）在每英寸的区域内能够打印的点的数量。较高的DPI值意味着更多的点被打印在每英寸的区域内，从而提供更高的打印分辨率和更清晰的图像，一般在设计出图后打印会显示dpi
  • PPI（每英寸像素数）指的是显示设备（如电脑显示器或手机屏幕）在每英寸的区域内能够显示的像素数量。较高的PPI值意味着更多的像素被显示在每英寸的区域内，从而提供更高的显示分辨率和更清晰的图像。
  • 在打印和图像设计中，DPI常用于描述打印质量和图像分辨率，而PPI常用于描述显示设备的分辨率，但在电子屏幕上，每英寸点数和每英寸像素数是一样的（基本单位长宽相等为方形），所以此时dpi就是ppi，即ppi是dpi在电子设备上应用的延申，例如，当我们说一个图像的分辨率是300 DPI时，意思是在每英寸的区域内有300个点或像素。
  1. 屏幕密度计算
  • 屏幕密度：√（水平像素数² + 垂直像素数²）/ 屏幕尺寸
//根据显示设备分辨率PPI计算，DPI同理
let width = 1080;  //屏幕宽度
let height = 1920;  //屏幕高度
let size = 5.0;   // 手机尺寸（对角距离按英寸算）
let data = Math.sqrt(Math.pow(width,2) + Math.pow(height,2))
let ppi = Math.floor(data / size)
  • dp: 即dip，安卓专用，1dp在屏幕像素密度为160ppi/dpi时为1px长度。dp= px / (dpi / 160)
    • dp以ppi为160为基准，1dp = 1px，ppi为320，1dp = 2px，以此类推
  • sp：字体大小单位，设置字体大小，比如在手机设置中调整字体大小，实际上调整的是sp，然后通过sp和px做换算
  • pt：有两种，一种是用于印刷行业，1pt=1/72英寸在ps和word中用于打印时中文单位为磅；另一种是ios开发中专用的最小长度单位，当ppi为163时，1pt=1px
  • 问题：既然有px了，为啥还需要dp、pt？这是因为px没有固定的物理长度，1px在同样尺寸的物理屏幕上，pt和px的换算不同，那1px显示的效果就不同。
  • 如何做屏幕适配：如果拿到手的图片以px为单位（其实没有如果），一般情况下会有一倍图、二倍图等文件字样，在ppi为163或160时采用一倍图，ppi为326或320时采用二倍图，以此类推
UI设计尺寸规范

相关信息

注意：此模块尺寸规范仅供参考

  • 画布大小为375 * 812 pt，相关模块高度设计
    • 状态栏：375 * 48 pt
    • 导航栏：375 * 44 pt
    • 标签栏：375 * 49 pt
    • 按钮：最大高度44 pt， 最小高度40 pt
    • 搜索框：常用高度为29 - 33 pt
    • 大标题所处区域：高度最小40 pt
    • 卡片页竖向边距：最小8pt，常用10 - 20 pt
    • 页边距：12 - 18 pt
  • 字体大小规范
    • 导航栏标题 / 详情页标题：字号18
    • 大标题、按钮文字：字号16
    • 小标题、正文文字：字号13-15
    • 辅助性提示文字：字号11-12
    • 底部图标栏文字：字号10
    • 标签组件文字：字号15-17
    • 功能入口文字：字号12-13
  • 文字层级颜色规范：37A
    • 重要级文字信息、标题信息：#333333
    • 普通级文字信息：#777777
    • 辅助、次要文字信息：#AAAAAA
    • 背景色：#EEEEEE - #F8F8F8
  • 启动和功能图标尺寸（单位px）
    • 安卓：48、72、96、144、192、512
    • Ios：长宽120，圆角半径25
