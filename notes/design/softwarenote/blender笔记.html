Blender笔记

禹穆穆2023/10/1...大约 7 分钟设计类笔记

常用快捷键

物体/编辑模式通用

  • 界面平移：Shift+鼠标中键
  • 界面缩放： 鼠标滚轮
  • 界面旋转： 鼠标中键拖到
  • 正视图（后）： 数字1（Ctrl）
  • 右视图（左） ：数字3（Ctrl）
  • 顶视图（底） ：数字7（Ctrl）
  • 进/退四格视图： Ctrl + Alt + Q
  • 沿x/y/z轴移动m个单位： G + x/y/z + 数字m/鼠标拖动
  • 物体沿x/y/z轴旋转m度： R + x/y/z + 数字m/鼠标拖动
  • 物体沿x/y/z轴缩放m单位： S + x/y/z + 数字m/鼠标拖动
  • 物体整体缩放： S
  • N面版（观察调节变换信息/摄像机等）： N
  • 物体移动/旋转/缩放复位： Alt + G/R/S
  • 复制物体并进入Grab模式： Shift + D
  • 添加物体： Shift + A
  • 删除物体： X / delect
  • 原位复制物体： ctrl C V
  • 返回上一步： Ctrl + Z
  • 游标目视定位： Shift + 鼠标左键
  • 游标吸附定位： 进编辑模式后点模式>网格>吸附>游标到选中项
  • 游标恢复到坐标系中心：Shift + C
  • 改变物体原点： 游标吸附定位到目标位置，右键>设置原点>几何中心到原点
  • 透视： Alt + Z
  • 微调版面： F9
  • 注意：鼠标悬停在按钮上按住F1可以打开在线帮助

物体模式

  • 合并物体 — Ctrl + J
  • 剪裁视图 — Alt+ B

编辑模式

  • 进入/退出编辑模式 — Tab
  • 加/减/全选 — Shift/ Ctrl/ A
  • 点/线/面模式 — Q上面的1/2/3
  • 挤出 — E
  • 内插面 — i
  • 小刀工具 — K（配合Shift）
  • 法向翻转 — Alt + N
  • 填充面 — Alt + F
  • 倒角— Ctrl + B
  • 偏移）环切边 —（Shift +）Ctrl + R
  • 循环边 — Alt + 鼠标左键
  • 并排边/循环面 — Ctrl + Alt + 鼠标左键
  • 法向收缩 — Alt + S
  • 顶点滑移 — GG / Shift + V
  • 合并顶点 — M
  • 连接顶点 — J / F（推荐J）
  • 隐藏顶点 — H（Alt+L恢复）
  • 选择相连元素 — L

操作细节

物体模式

  • 物体的各种变换都在微调版面可以调整，同时在N版面也可以调整物体位置及尺寸的具体参数
  • 物体的原点和游标不一样，物体的变换是根据原点来的
  • cavity 滤色显示：着色方式里勾选。可以在物体模式下让线面具体显示

编辑模式

  • 各种操作的微调版面很重要，比如两个面做内插面是微调版面中勾选各面和选择外侧
  • 建模建议使用四边形做（算法计算时找的是对边，其他形状对边没有或者不好找有多个），不推荐多边形，三角形少用。
  • 尖分面是从面的中心和各顶点相连分割面，面三角化是把各顶点相连均匀分成三角形
  • 翻转面的法向：网格->法向->法向
  • 填充面：栅格填充（要求偶数边，填充后是四边形）和填充
  • 细分在边和面模式里就是分割，配合微调版面；融并边和融并面都可以合并面，融并顶点可以在不改变结构的情况下删除顶点，删除顶点会删除的很干净
  • 倒角对点线面都可以使用
  • 桥接循环边：边->桥接循环边。将两组边数相同的循环边连成面。应用：墙体打洞
  • 顶点滑移时可以打开吸附顶点；合并点可以打开自动合并，选项->勾选自动合并，或者暴力合并，合并顶点->按距离
  • 衰减编辑：比如拉起一个顶点其四周的边线会很柔和。按钮在磁铁旁边，可以改变衰减方式（就是柔和程度），比如应用是做地形
  • 选择可以用间格式弃选
  • 拆分和分离：都在网格中，拆分是在一个物体里拆分，分离是把一个物体分成两个物体
  • 曲线下的点可以用挤出或者细分增加点，可以在添加里加入参考图/背景，用钢笔工具描线，曲线插件extra bool形状更多，弹簧等形状都有，曲线和网格可以互相转化（全选右键->转化到），u分辨率（在右下方纹理按钮的上面）决定了曲线的平滑程度；曲线也有方向，打开法向可以看见，曲线的半径（N版面调整）决定曲线顶点的粗细
  • 把曲线弄成管子/绳子等，用倒角（在和u分辨率的版面上），结合n版面的参数调整；改变其横截面时需要将横截面形状画出来，然后在版面的倒角->物体里用吸管吸一下（类似C4D的扫描）；将横截面做出来后新建圆环吸附，可以做出一个有规则的圆体物（将横截面旋转一周）
  • 晶格，用于定义一块扭曲的空间，可以在属性里修改，也可以进编辑模式对点线进行操作，常常配合晶格修改器

修改器

  • 修改器是在物体模式下使用的，但也可以在编辑模式下查看，在修改器中可以修改是否查看，应用修改器后修改器和模型会合为一个；修改器可以反复修改；所有修改器基本都是对顶点做调整，注意提前细分
  • 多个修改器的顺序可以修改，名字可以改，同类型的修改器可以添加多个
  • 阵列修改器偏移：物理偏移（可以添加一个空物体标记目标偏移位置，在修改器里用吸管吸空物体，位置、缩放、旋转的偏移，可以做出环形、渐大渐小等阵列效果）
  • 布尔常用于硬表面建模；线框修改器常用来做网状结构（铁丝网；分子结构等，在修改器前可以用一下细分，反细分），但一定要有面，只有点线无效果
  • 表面细分修改器主要做了两件事，细分和过度。细分是将一个面切分为4^n个小面，和cpu性能有关；过渡是调整面的法向，平滑过渡。配合右键的边线折痕，可以做折痕效果；表面细分对三角面和多边面布线会有瑕疵，要使用四边形
  • 曲线修改器可以让物体按照曲线的路径变形，注意给物体分段，曲线移动的是顶点，物体原点和曲线原点建议一致，曲线的半径可以影响到物体的粗细

材质和纹理

  • 关联材质和把材质拖到物体表面没区别；复制后的材料不关联原材质；没有被使用过的材质不会被保存
  • 复杂材质通常是使用材质节点制作的
  • UV纹理
    • 坐标表示：u是横坐标，v是纵坐标。在N版面可以查看顶点的坐标
    • uv映射可以自由指定（就是正视图顶点和uv展开图顶点不是顺序对应的，是自由的，可以旋转、移动、缩放面改变）
    • uv展开会自动把每个面布局的很规整，在编辑模式下选择面可以显示对应的uv面（勾选生成uv）
    • 绘制uv贴图流程：导出uv布局图->PS里打开->绘制各个面->隐藏布局图->保存psd格式->添加材质、贴图->修改贴图后，图像里的重载，刷新显示
    • 基本体的模型自动展开uv很简单，但往往很多模型不是基本体，需要手动展开uv。
    • 手动展开方法：<a> 全选->智能展开uv（在点线面模式那排的uv里），此方法不推荐。<b>选中要剪开的边，标记为缝合边（在uv里，相当于剪刀，同样也可以清除缝合边）->全选后展开->不断重复以上两部直到每个面清晰全面而且易于绘制 ->调整布局（相连面的横竖方向、孤岛等，变换工具和L键在uv展开图里也适用）
    • uv对齐：对x/y对齐（在uv编辑器里的uv按钮->对齐）；孤岛对齐（在uv编辑器里的N版面里手动调整，可以打开磁铁移动时会吸附顶点）；沿活动四边面展开（需要整体的一角为90度且两边和xy轴分别平行）；缝合和拆分（会将相连的面切开成孤岛，或把孤岛合并）
