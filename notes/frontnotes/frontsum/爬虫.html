Crawlee.js

初识

  • 官方文档：https://crawlee.dev/docs/introduction
  • 开源
  • 支持js和ts
  • Node最低版本Node.js 16.0 版

安装和首次启动

  • 安装
pnpm install crawlee palywright puppeteer
npx crawlee install
  • 编写爬虫脚本
// index.js
import { CheerioCrawler } from 'crawlee';

const crawler = new CheerioCrawler({
    maxRequestsPerCrawl: 20,
    async requestHandler({ $, request, enqueueLinks }) {
        const title = $('title').text();
        console.log(`The title of "${request.url}" is: ${title}.`);
        await enqueueLinks();
    },
});

await crawler.run(['https://crawlee.dev']);
  • node运行脚本
node index.js

主要爬虫类

  • Crawlee 带有三个主要的爬虫类：CheerioCrawler、PuppeteerCrawler 和 PlaywrightCrawler。
    • CheerioCrawler：使用 Cheerio 库来解析 HTML 并提取数据，但无法处理 JavaScript 渲染。
    • PuppeteerCrawler：使用 Puppeteer 库来控制浏览器并执行 JavaScript。
    • PlaywrightCrawler：Playwright 是 Puppeteer 的更强大、功能更齐全的继任者。它可以控制 Chromium、Chrome、Firefox、Webkit 和许多其他浏览器。如果您还不熟悉 Puppeteer，并且需要一个无头浏览器，请使用 Playwright。
