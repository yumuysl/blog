思维导图:

一、three.js安装及项目文件夹目录

文件目录

新建一个文件夹dome，打开终端通过下方npm方式安装three.js，把所有代码写在一个文件里维护起来很麻烦，所以按照以下目录新建文件：

├─public
│  └─style
│          main.css
└─src
    │  main.js
    ├─assets
    └─world
        │  World.js
        │
        ├─components
        │      camera.js
        │      cube.js
        │      scene.js
        │
        └─systems
                renderer.js
                Resizer.js

在dome下新建main.html文件，内容如下：

<!DOCTYPE html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Three.js | 入门练习</title>
  <link type="text/css" rel="stylesheet" href="./public/style/main.css" >
</head>

<body>
  <div id="scene-container">
  </div>
</body>
<script type="importmap">
	{
	"imports": {
		"three": "../../../node_modules/three/build/three.module.js",
		"three/examples/jsm/controls/OrbitControls": "../../../node_modules/three/examples/jsm/controls/OrbitControls.js"
	}
	}
</script>
<script type="module" src="./src/main.js" ></script>
</html>

其中，css文件路径为public/style/main.css，内容如下：

body {
  margin: 0;
  overflow: hidden;
  width: 100%;
  height: 100%;
}

#scene-container {
  position: absolute;
  width: 100%;
  height: 100%;
}

到这儿ok，当然文件目录可按照合适自己的方式修改，到这儿项目还不能运行，我们要在具体的js文件中写入内容。

通过npm方式安装

注

也可以通过CDN方式安装，但考虑到以后长时间用，建议npm方式安装

在文件夹内打开cmd输入下面命令：

npm init
npm install three

然后导入到项目中，如:

import * as THREE from "three";

一般来说，项目中不会所有的模块都会用到，所以我们不会在每个文件里都导入所有的模块，按需导入即可。

二、第一个立方体

欲让画面中显示目标物体，我的理解是需要经过五个环节，即场景、相机、模型、光照和渲染，场景是相机和模型的载体，相机是类似于我们人眼看东西的视角，模型是我们的显示事物的本体，经常是需要光照才可以看见，渲染是在相机的视角内把准备好的模型按照要求显示的最后环节。总体来说，和三维软件如blender、C4D、3Dsmax等有共同之处，如果你使用过相关三维软件，相信理解会更深入。这里为了方便演示，光照环节我们在第三节会有。

  • 先说api：

创建场景

创建场景的方法如下：

import { Scene, Color } from "three";
let scene = new Scene()

//此步可省
scene.background = new Color('skyblue');

这样就创建好了场景，一般来说，一个项目只需要一个场景，在创建中可以添加相机和模型。

创建相机

在有了场景后，接下来就是创建相机了，或者是模型，创建相机和模型顺序不是很重要，但建议优先创建相机，养成习惯。相机有两种，一种是透视相机，这种相机很符合我们人眼看到的，有各个角度的透视（就是远近、虚实关系），另一种是正交相机，这种相机显示的物体用人眼看是平的，可以理解为每个部分的距离都相等，我们这里只谈透视相机，正交相机我们后续会了解到的。创建透视相机的方法如下：

import { PerspectiveCamera } from "three";

let fov = 35;  //景深，可以理解为画面的远近
let aspect = 1;   //窗口纵横比
let near = 0.1;   //最近距离
let far = 100;    //最远距离，超出最近距离或者最远距离的部分相机的盲区，不可见。
let carmar = new PerspectiveCamera(fov, aspect, near, far);

这样我们就创建了一个透视相机，接下来我们继续创建模型。

创建物体模型

模型需要两个部分，一个是形，一个是材质。形就是物体模型的形状，可以理解为不带材质的任何物体，桌子、杯子、房子等等，这部分可以通过三维软件建模得到，也可以获取现成的三维模型；另外就是材质，材质可以是纯色或渐变色，也可以是从现实生活中提取的各种纹理，亦或是各种图画等等。创建模型的方法如下：

import { BoxBufferGeometry, Mash, MashbasicMaterial } from "three";

let geometryCube = new BoxBufferGeometry();  //形
let material = new MashbasicMaterial();   //材质，为了方便演示这里采用不同照明也可以看见的基本材料，需要光照可见的是PBR标准材料
let cube = new Mesh(geometryCube, material);  //创建物体模型

接下来我们创建渲染器。

创建渲染器

在这一步，我们可以按照自身需要渲染画面，在这里我们的目标是渲染一个简单的立方体，创建渲染器方法如下：

import { WebGLRenderer } from "three";

let renderer = new WebGLRenderer();

到这儿，我们已经和常用api打了个脸熟，接下来我们来讨论具体的操作。

完整具体的操作

main.js文件中：

import { World } from "./world/World.js";

function main(){
  const container = document.querySelector("#scene-container");
  const world = new World(container);

  world.start();

}

main();

World.js文件中：

import { createCamera } from './components/camera.js';
import { createCube } from './components/cube.js';
import { createScene } from './components/scene.js';
import { createLights } from './components/lights.js';

import { createRenderer } from './systems/renderer.js';
import { Resizer } from './systems/Resizer.js';

class World{
  //封装私有属性
  #camera;
  #renderer;
  #scene;

  //初始化
  constructor(container){

    this.#camera = createCamera();
    this.#scene = createScene();
    this.#renderer = createRenderer();
    
    container.append(this.#renderer.domElement);
    const cube = createCube();

    this.#scene.add(cube);

    //创建resizer实例
    const resizer = new Resizer(container, this.#camera, this.#renderer);

  }

  //渲染
  render(){
    this.#renderer.render(this.#scene, this.#camera);
  }
}

export { World };

components/camera.js中:

import { PerspectiveCamera } from "three";

//创建透视相机
function createCamera(){
  const camera = new PerspectiveCamera(35,1,0.1,100);

  camera.position.set(2,2,10);

  return camera;
}

export { createCamera };

components/cube.js中：

import { BoxBufferGeometry, CircleBufferGeometry, Mesh, MeshStandardMaterial } from "three";

//创建立方体
function createCube(){
  const geometryCube = new BoxBufferGeometry(2,2,2);

  const material = new MeshBasicMaterial();

  const cube = new Mesh(geometryCube, material);

  cube.position.set(0,0,0);
  //旋转立方体
  cube.rotation.set(0,0,0);
  return cube;
}

export { createCube };

components/scene.js中：

import { Scene, Color } from "three";

//创建场景
function createScene(){
  const scene = new Scene();
  scene.background = new Color('skyblue');
  return scene;
}

export { createScene };

systems/renderer.js中：

import { WebGLRenderer } from "three";

//创建渲染系统
function createRenderer(){
  const renderer = new WebGLRenderer();
  return renderer;
}

export { createRenderer };

systems/Resizer.js中：

到这儿我们就可以运行整个项目了，点击index.html入口文件没有意外就可以看到一个静止的立方体了。
