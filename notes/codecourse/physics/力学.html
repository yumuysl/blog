质点运动学

位移、速度、加速度

位置矢量（位矢）

  • 建立直角坐标系后，物体的位置可用坐标A(x, y, z)表示，向量r⃗=OA⃗\vec{r} = \vec{OA}r=OA称为位置矢量，也记为r⃗=xi⃗+yj⃗+zk⃗\vec{r} = x\vec{i} + y\vec{j} + z\vec{k}r=xi+yj​+zk

运动函数

  • 运动函数就是质点坐标x, y. z随时间t的变化关系，将t消掉，可判断物体沿什么轨迹运动

位移和路程

  • 位移Δr⃗\Delta\vec{r}Δr，用起点到终点的有向线段表示，矢量
  • 路程：Δs\Delta sΔs：物体实际运动的路径长，标量
  • 辨析：Δr⃗\Delta\vec{r}Δr是位移，∣Δr⃗∣|\Delta\vec{r}|∣Δr∣是位移的大小，Δs\Delta sΔs是物体到原点距离的变化，只有ds=∣dr⃗∣ds = |d\vec{r}|ds=∣dr∣是对的，Δs≠∣Δr⃗∣\Delta s \neq |\Delta\vec{r}|Δs=∣Δr∣

速度

  • 平均速度v⃗=Δr⃗Δt⃗\vec{v} = \frac{\Delta\vec{r}}{\Delta\vec{t}}v=ΔtΔr​，平均速率是路程除以时间v⃗=ΔsΔt\vec{v} = \frac{\Delta s}{\Delta t}v=ΔtΔs​
  • 瞬时速度就是对每个坐标分量求导

v⃗=dr⃗dt=dxdti⃗+dydtj⃗+dzdtk⃗\vec{v} = \frac{d\vec{r}}{dt} = \frac{dx}{dt}\vec{i} + \frac{dy}{dt}\vec{j} + \frac{dz}{dt}\vec{k} v=dtdr​=dtdx​i+dtdy​j​+dtdz​k

  • 瞬时速率是瞬时速度的大小，速度大小

v=∣dr⃗dt∣=dsdt=(dxdt)2+(dydt)2+(dzdt)2v = |\frac{d\vec{r}}{dt}| = \frac{ds}{dt} = \sqrt{(\frac{dx}{dt})^2 + (\frac{dy}{dt})^2 + (\frac{dz}{dt})^2} v=∣dtdr​∣=dtds​=(dtdx​)2+(dtdy​)2+(dtdz​)2​

加速度

  • 加速度是速度的变化率，具体而言：
    • 平均加速度a⃗=Δv⃗Δt\vec{a} = \frac{\Delta\vec{v}}{\Delta t}a=ΔtΔv​
    • 瞬时加速度就是对每个速度分量求到a⃗=dv⃗dt\vec{a} = \frac{d\vec{v}}{dt}a=dtdv​
    • 注意：加速度与速度同号为加速、异号为减速，而不是正数为加速、负数为减速

位移、速度、加速度的互求

  • 求导；坐标x,y,z -> 速度 -> 加速度

v=dxdt，a=dvdtv =\frac{dx}{dt}，a = \frac{dv}{dt} v=dtdx​，a=dtdv​

  • 积分：加速度 -> 速度 -> 坐标

v=v0+∫0tadt,x=x0+∫0tvdtv = v_0 + \int_0^t a dt, x = x_0 + \int_0^t v dt v=v0​+∫0t​adt,x=x0​+∫0t​vdt

加速度的分解

  • 加速度可以沿着速度方向分解
    • 与速度方向平行的称为切向加速度，改变速度的大小at=dvdta_t = \frac{dv}{dt}at​=dtdv​
    • 与速度方向垂直的称为法向加速度，改变速度的方向an=v2ρa_n = \frac{v^2}{ρ}an​=ρv2​（ρ为曲率半径）

圆周运动参数的互求

  • 求导：角位移θ -> 角速度ω -> 角加速度α

ω=dθdt，α=dωdtω = \frac{dθ}{dt}，α = \frac{dω}{dt} ω=dtdθ​，α=dtdω​

  • 积分：角位移θ <- 角速度ω <- 角加速度α

ω=ω0+∫0tαdt，θ=θ0+∫0tωdtω = ω_0 + \int_0^t α dt，θ = θ_0 + \int_0^t ω dt ω=ω0​+∫0t​αdt，θ=θ0​+∫0t​ωdt

角量和线量的关系

  • 线速度：v=ωRv = ωRv=ωR
  • 法向加速度an=v2R=ω2Ra_n = \frac{v^2}{R} = ω^2 Ran​=Rv2​=ω2R
  • 切向加速度at=dvdt=αRa_t = \frac{dv}{dt} = αRat​=dtdv​=αR
  • 向心力：F=mv2RF= \frac{mv^2}{R}F=Rmv2​

常见的力

重力

  • 公式：G=mgG=mgG=mg

  • 公式：F=kxF=kxF=kx，（以弹簧为例，k为弹性系数，x为形变量）

  • 静摩擦力：f≤μsNf\leq μ_s Nf≤μs​N，与接触面平行，指向阻碍相对运动的方向
  • 静摩擦因数μsμ_sμs​略大于动摩擦因数μμμ

  • 公式：F=GMmR2F=G\frac{Mm}{R^2}F=GR2Mm​
  • 方向：指向两个物体的连线方向

牛顿运动定律

牛顿第一定律

  • 定义；任何物体都保持静止或匀速直线运动的状态，除非作用在它上面的力迫使它改变这种状态

牛顿第二定律

  • 定义：F⃗=dp⃗dt\vec{F}=\frac{d\vec{p}}{dt}F=dtdp​​，若质量不变，则F⃗=ma⃗=mdv⃗dt\vec{F}=m\vec{a}=m\frac{d\vec{v}}{dt}F=ma=mdtdv​

牛顿第三定律

  • 定义：F12⃗=−F21⃗\vec{F_{12}} = -\vec{F_{21}}F12​​=−F21​​
