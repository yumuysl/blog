树

禹穆穆2023/4/20...大约 11 分钟数据结构

生活中的树由节点和枝干组成，但我们这里讨论的树是倒过来的树，根在最顶端，而最顶端的节点称为根节点，一个节点可以连接多个其他节点，我们本节主要讨论的是二叉树，而二叉树最多只有两个子节点，即左子节点和右子节点，每个节点由本身的值和指向左子节点和右子节点的两个指针组成。

一、二叉树

（一）常用术语

  1. 父节点和子节点：A节点的左指针或右指针指向B节点，那么A节点是B节点的父节点，B为A节点的子节点，父节点和子节点是相对的，一个节点可以是子节点也可以是其他节点的父节点；
  2. 根节点：无父结点，位于最顶端；
  3. 叶节点：无子节点，位于树结构的最底层
  4. 节点的层：从根节点开始，到底层结束，依次递增，根节点的层为1
  5. 边：逻辑图上连接两个节点的线段
  6. 节点的高度：从最远节点到该节点的边的数量

（二）分类

  1. 满二叉树（完美二叉树）：节点的层数为n，则满二叉树的最底层节点数量为2^n^，所以总节点的数量为2^n+1^-1个
  2. 完全二叉树：除了最底层节点未被填满，而且最底层节点靠左
  3. 完满二叉树：除了最底层外，其余所有节点都有两个子节点
  4. 平衡二叉树：任意节点的左子树和右子树的高度差其绝对值不超过1

（三）遍历（递归或循环）

  1. 广度优先遍历：按照层的顺序遍历，从左往右。父节点 > 左子节点 > 右子节点
/* 层序遍历 */
function levelOrder(root) {
    // 初始化队列，加入根节点
    const queue = [root];
    // 初始化一个列表，用于保存遍历序列
    const list = [];
    while (queue.length) {
        let node = queue.shift(); // 队列出队
        list.push(node.val); // 保存节点值
        if (node.left) queue.push(node.left); // 左子节点入队
        if (node.right) queue.push(node.right); // 右子节点入队
    }
    return list;
}
  1. 深度优先遍历
    • 前序遍历：根节点/父节点 > 左节点（从最底层开始） > 右节点（从最底层开始），区别于广度优先遍历
    • 中序遍历：左节点 > 父节点 > 右节点
    • 后序遍历：左节点 > 右节点 > 父节点
    /* 前序遍历 */
 function preOrder(root) {
     if (root === null) return;
     // 访问优先级：根节点 -> 左子树 -> 右子树
     list.push(root.val);
     preOrder(root.left);
     preOrder(root.right);
 }
 
 /* 中序遍历 */
 function inOrder(root) {
     if (root === null) return;
     // 访问优先级：左子树 -> 根节点 -> 右子树
     inOrder(root.left);
     list.push(root.val);
     inOrder(root.right);
 }
 
 /* 后序遍历 */
 function postOrder(root) {
     if (root === null) return;
     // 访问优先级：左子树 -> 右子树 -> 根节点
     postOrder(root.left);
     postOrder(root.right);
     list.push(root.val);
 }

二、二叉搜索树

添加

二叉搜索树的添加操作较为简单，我们这里搜索采用稍微复杂的方式，用于理解递归。

class Node{
  constructor(data){
    this.value = data;
    this.left = null;
    this.right = null;
  }
}

//二叉树
class BinaryTree{
  #root;
  #length = 0;

  //第一种查找方式（有些繁琐）：递归
  #findwhere(data, current){
    if(data > current.value){
      //右节点:1
      if(current.right != null){
        return this.#findwhere(data, current.right);
      }else{
        return [current,2];
      }
    }else if(data < current.value){
      //左节点:0
      if(current.left != null){
        return this.#findwhere(data, current.left);
      }else{
        return [current,1];
      }
    }else{
      return [current,0];
    }
  }

  //添加节点
  insert(nodeValue){
    if(this.#length == 0){
      this.#root = new Node(nodeValue);
    }else{
      let node = new Node(nodeValue);
      let current = this.#findwhere(node.value, this.#root);
      if(current[1] == 1){
        current[0].left = node;
      }else if(current[1] == 2){
        current[0].right = node;
      }else{
        return "数值重复";
      }
    }
    this.#length++;
    return "已添加";
  }
}

删除

二叉搜索树的删除操作分为三种情况，即节点的子节点为0，1，2。当子节点数量为0或1时，直接替换即可，当子节点数量为2时，用该节点中序遍历的下一个节点替换。

class Node{
  constructor(data){
    this.value = data;
    this.left = null;
    this.right = null;
  }
}

//二叉树
class BinaryTree{
  #root;
  #length = 0;

  //添加节点
  insert(nodeValue){...}

  //删除节点
  remove(data){
    let current = this.#root;
    let prover = null;

   //第二种查找方式：循环（但要注意break）
    while(current !== null){
      if(current.value === data)  break;
      prover =  current;   //标记前一个节点
      if(current.value < data){
        current = current.right;
      }else{
        current = current.left;
      }
    }

    //console.table(prover)
    //当current子节点为0或1时
    if( current.left == null || current.right == null ){
      let child = current.left == null ? current.right : current.left;
      if(current != this.#root){
        if(prover.left == current){
          prover.left = child;
        }else{
          prover.right = child;
        }
      }else{
        this.#root = current;
      }
    }else{ //当current子节点为2时，找到中序遍历该节点的下一个节点
      let midOrderNode = current.right;
      while(midOrderNode != null){
        if(midOrderNode.left == null) break;
        midOrderNode = midOrderNode.left;
      }
      this.remove(midOrderNode.value);
      current.value = midOrderNode.value;
      this.#length++;     // 由于上面调用remove递归会重复减一，但实际上是替换，加1减2
    }

    if(this.#length >0){
      this.#length --;
    }
  }

}

采用中序遍历可以得到升序序列。

class Node{
  constructor(data){

    this.right = null;
  }
}

//二叉树
class BinaryTree{
  #root;
  #length = 0;

  //中序遍历
  #midOrder(root,callback){
    if(root !== null){
      this.#midOrder(root.left,callback);
      callback(root.value);
      this.#midOrder(root.right,callback);
    }
    
  }

  //添加节点
  insert(nodeValue){...}


  //排序：通过中序遍历可以得到升序列表
  sort(){
    let dataArray = [];
    this.#midOrder(this.#root,function(value){
      dataArray.push(value);
    });
    return dataArray;
  }
  
}

三、AVL树

  • 当二叉树在增加或者减少节点时会发生二叉树退化为链表的形式，这是复杂度由原来的O(logsub(n))变为O(n)，这不是我们想要的，所以诞生了AVL树，通过约束左右节点保持原有的复杂度。
  • 而黑红树属于AVL树，但他比AVL树松懈，更加灵活，比AVL树受欢迎。

节点高度和平衡因子

节点高度

  • 节点的高度：当前节点到叶节点的距离
  • 用代码实现获取和更新节点高度：
  //获取节点高度
  #getheight(node){
    return node == null ? -1 : node.height;
  }

  //更新节点高度
  #updateHeight(node){
    node.height = Math.max(this.height(node.left), this.height(node.right)) + 1;
  }

平衡因子

  • 平衡因子：当前节点左节点和右节点的高度差，AVL树平衡因子绝对值不大于1。
  • 为什么需要平衡因子：方便处理二叉树平衡问题，当平衡因子大于1时说明此二叉树即将退化为链表，要如愿保持原来的复杂度需要处理左右节点
  • 获取平衡因子的代码：
  //获取节点平衡因子：平衡因子 = 左节点高度 - 右节点高度，要求平衡因子绝对值小于等于1
  #getBalaceFoc(node){
    if(node == null){
      return 0;
    }
    return  this.#height(node.left) - this.#height(node.right);
  }

右旋

当结点左节点高度大于右边时，需要将左边的节点向右转一下，代码如下：

  //右旋
  #rightTurn(node){
    let leftChild = node.left;
    let grandChild = leftChild.right;
    leftChild.right = node;
    node.left = grandChild;

    //更新节点高度
    this.#updateHeight(node);
    this.#updateHeight(leftChild);

  }

左旋

同理，当结点右节点高度大于左边时，需要将右边的节点向左转一下，代码如下：

  //左旋
  #leftTurn(node){
    let rightChild = node.right;
    let grandChild = rightChild.left;
    rightChild.left = node;
    node.right = grandChild;

    //更新节点高度
    this.#updateHeight(node);
    this.#updateHeight(rightChild);
  }

AVL树的典型应用

  • 用于存储大型数据，适合高频查找、低频增删的场景

AVL树增加节点时如何平衡？代码如下：

// AVL树节点
class AVLNode {
  constructor(value, left = null, right = null) {
    this.value = value;
    this.left = left;
    this.right = right;
    this.height = 1;
  }
}

// AVL树
class AVLTree {
  constructor() {
    this.root = null;
  }

  // 获取节点高度
  getHeight(node) {
    if (!node) return 0;
    return node.height;
  }

  // 获取节点平衡因子
  getBalanceFactor(node) {
    if (!node) return 0;
    return this.getHeight(node.left) - this.getHeight(node.right);
  }

  // 右旋
  rotateRight(node) {
    const left = node.left;
    const rightOfLeft = left.right;
    left.right = node;
    node.left = rightOfLeft;
    node.height = Math.max(this.getHeight(node.left), this.getHeight(node.right)) + 1;
    left.height = Math.max(this.getHeight(left.left), this.getHeight(left.right)) + 1;
    return left;
  }

  // 左旋
  rotateLeft(node) {
    const right = node.right;
    const leftOfRight = right.left;
    right.left = node;
    node.right = leftOfRight;
    node.height = Math.max(this.getHeight(node.left), this.getHeight(node.right)) + 1;
    right.height = Math.max(this.getHeight(right.left), this.getHeight(right.right)) + 1;
    return right;
  }

  // 先左旋后右旋
  rotateLeftRight(node) {
    node.left = this.rotateLeft(node.left);
    return this.rotateRight(node);
  }

  // 插入节点
  insert(value) {
    this.root = this._insert(this.root, value);
  }

  _insert(node, value) {
    if (!node) return new AVLNode(value);
    if (value < node.value) {
      node.left = this._insert(node.left, value);
    } else if (value > node.value) {
      node.right = this._insert(node.right, value);
    } else {
      return node;
    }
    node.height = Math.max(this.getHeight(node.left), this.getHeight(node.right)) + 1;
    const balanceFactor = this.getBalanceFactor(node);
    if (balanceFactor > 1 && value < node.left.value) {
      return this.rotateRight(node);
    }
    if (balanceFactor < -1 && value > node.right.value) {
      return this.rotateLeft(node);
    }
    if (balanceFactor > 1 && value > node.left.value) {
      return this.rotateLeftRight(node);
    }
    return node;
  }
}

/**
 * 树结构工具函数
 * 提供数组转树和树转数组的功能
 */

/**
 * 通用树节点接口
 */
export interface TreeNode {
  id: number | string;
  parentId: number | string | null;
  children?: TreeNode[];
  [key: string]: any;
}

/**
 * 数组转树结构函数
 * 将扁平数组转换为嵌套的树结构
 * @param items 扁平数组
 * @param options 配置选项
 * @param options.idKey id字段名，默认为'id'
 * @param options.parentIdKey 父id字段名，默认为'parentId'
 * @param options.childrenKey 子节点字段名，默认为'children'
 * @param options.rootId 根节点的父id值，默认为null
 * @returns 转换后的树结构数组
 */
export function arrayToTree<T extends Record<string, any>>(
  items: T[],
  options: {
    idKey?: keyof T;
    parentIdKey?: keyof T;
    childrenKey?: string;
    rootId?: any;
  } = {},
): TreeNode[] {
  const {
    idKey = "id",
    parentIdKey = "parentId",
    childrenKey = "children",
    rootId = null,
  } = options;

  // 创建id到节点的映射
  const itemMap = new Map<string, any>();
  // 存储根节点
  const rootItems: TreeNode[] = [];

  // 初始化映射
  items.forEach((item) => {
    const id = item[idKey];
    itemMap.set(id, { ...item, [childrenKey]: [] });
  });

  // 构建树结构
  items.forEach((item) => {
    const id = item[idKey];
    const parentId = item[parentIdKey];
    const currentItem = itemMap.get(id)!;

    if (parentId === rootId) {
      // 根节点
      rootItems.push(currentItem);
    } else {
      // 非根节点，添加到父节点的子节点列表中
      const parentItem = itemMap.get(parentId);
      if (parentItem) {
        parentItem[childrenKey].push(currentItem);
      }
    }
  });

  return rootItems;
}

/**
 * 树结构转数组函数
 * 将嵌套的树结构转换为扁平数组
 * @param tree 树结构数组
 * @param options 配置选项
 * @param options.childrenKey 子节点字段名，默认为'children'
 * @param options.includeChildren 是否包含子节点，默认为true
 * @returns 转换后的扁平数组
 */
export function treeToArray<T extends TreeNode>(
  tree: T[],
  options: {
    childrenKey?: keyof T;
    includeChildren?: boolean;
  } = {},
): T[] {
  const { childrenKey = "children", includeChildren = true } = options;

  const result: T[] = [];

  function traverse(node: T) {
    // 创建节点的副本，避免修改原节点
    const nodeCopy = { ...node };

    // 如果不包含子节点，则删除children字段
    if (!includeChildren && nodeCopy[childrenKey]) {
      delete nodeCopy[childrenKey];
    }

    result.push(nodeCopy);

    // 递归处理子节点
    if (
      node[childrenKey] &&
      Array.isArray(node[childrenKey]) &&
      node[childrenKey].length > 0
    ) {
      node[childrenKey]!.forEach((child: T) => traverse(child));
    }
  }

  // 遍历所有根节点
  tree.forEach((node) => traverse(node));

  return result;
}

/**
 * 深度优先搜索树结构
 * @param tree 树结构数组
 * @param callback 对每个节点执行的回调函数
 * @param options 配置选项
 * @param options.childrenKey 子节点字段名，默认为'children'
 * @param options.order 遍历顺序，'pre'为前序遍历，'post'为后序遍历，默认为'pre'
 */
export function traverseTree<T extends TreeNode>(
  tree: T[],
  callback: (node: T, index: number, parent: T | null) => void,
  options: {
    childrenKey?: keyof T;
    order?: "pre" | "post";
  } = {},
): void {
  const { childrenKey = "children", order = "pre" } = options;

  function dfs(node: T, index: number, parent: T | null = null) {
    if (order === "pre") {
      callback(node, index, parent);
    }

    if (node[childrenKey] && Array.isArray(node[childrenKey])) {
      (node[childrenKey] as T[]).forEach((child, childIndex) => {
        dfs(child, childIndex, node);
      });
    }

    if (order === "post") {
      callback(node, index, parent);
    }
  }

  tree.forEach((node, index) => dfs(node, index));
}

/**
 * 查找树中的节点
 * @param tree 树结构数组
 * @param predicate 判断函数，返回true表示找到目标节点
 * @param options 配置选项
 * @param options.childrenKey 子节点字段名，默认为'children'
 * @returns 找到的节点或undefined
 */
export function findNodeInTree<T extends TreeNode>(
  tree: T[],
  predicate: (node: T) => boolean,
  options: {
    childrenKey?: keyof T;
  } = {},
): T | undefined {
  const { childrenKey = "children" } = options;

  let result: T | undefined;

  function search(node: T): boolean {
    if (predicate(node)) {
      result = node;
      return true;
    }

    if (node[childrenKey] && Array.isArray(node[childrenKey])) {
      return (node[childrenKey] as T[]).some((child) => search(child));
    }

    return false;
  }

  tree.some((node) => search(node));
  return result;
}
